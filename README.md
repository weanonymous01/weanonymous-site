# We Anonymous | Fullstack App

A modern, dark-themed fullstack web application built with a focus on performance, clean UI, and conversion-driven design.

---

## Overview

Emergent is a sleek web app template designed for startups, SaaS products, and tech platforms. It uses a minimalist dark interface, modern typography, and a responsive layout to deliver a high-quality user experience.

---

## Features

* Dark-first user interface
* Fast and lightweight structure
* TailwindCSS-based styling
* Fully responsive design
* Component-based layout
* Conversion-focused sections
* Modern typography (Inter, Oswald, IBM Plex Mono)

---

## Tech Stack

* Frontend: HTML5, Tailwind CSS
* Styling: Custom CSS variables and utility classes
* Fonts: Google Fonts
* Design System: Minimal dark theme with accent color usage

## Getting Started

### 1. Clone the repository

```
git clone https://github.com/your-username/emergent-app.git
```

### 2. Open the project

Open `index.html` directly in your browser.

```
cd emergent-app
start index.html
```

## Customization

Theme variables can be adjusted in the root CSS:

```css
:root {
  --bg: #080808;
  --card: #111111;
  --accent: #D4FF3A;
  --text: #F5F5F5;
}
```
