CURRICULUM

6 Months. 6 Phases. One Clear Direction.

MONTH 01/01

Cyber Fundamentals and Setup

  • Networking basics and OSI model
  • Linux terminal and Bash scripting
  • Cryptography and encoding fundamentals
  • Passive and active reconnaissance
MONTH 02/02

OSINT and Social Engineering

  • OSINT framework and email enumeration
  • Social media intelligence gathering
  • Phishing simulation and pretexting
  • Account penetration concepts
MONTH 03/03

Network and Device Auditing

  • Nmap mastery and scripting engine
  • Vulnerability scanning with OpenVAS and Nessus
  • Wi-Fi auditing with Aircrack-ng
  • MITM attacks and detection in lab
MONTH 04/04

System Analysis and Exploitation

  • Metasploit framework end to end
  • Privilege escalation on Windows and Linux
  • Password cracking with Hashcat and John the Ripper
  • Forensics and log analysis basics
MONTH 05/05

Android and Mobile Security

  • APK structure and static analysis with JADX
  • Dynamic analysis using Frida and Objection
  • Insecure storage and communication flaws
  • OWASP Mobile Top 10 walkthrough
MONTH 06/06

Full Penetration Testing and Cert Prep

  • OWASP Top 10 and web application attacks
  • Complete penetration testing methodology
  • Bug bounty on HackerOne and Bugcrowd
  • Capstone black-box pen test and final report
